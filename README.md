# Iris Flower Classification Using Decision Tree

## Project Overview
This project demonstrates a supervised machine learning classification task using the Iris dataset and a Decision Tree classifier.

## Objective
- Load and understand a dataset
- Split data into training and testing sets
- Train a classification model
- Evaluate model performance

## Technologies Used
- Python
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
The project uses the Iris dataset available in Scikit-learn.

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Setosa
- Versicolor
- Virginica

## Algorithm Used
Decision Tree Classifier

## Results
- Accuracy: 100%
- Classification Report generated
- Confusion Matrix generated
- Feature Importance analysis completed

## Project Outputs
- confusion_matrix.png
- feature_importance.png

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
